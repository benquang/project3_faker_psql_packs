# GENERATE DATA using Faker Project

## I. Overview
Random choice, faker packs, list of tuples, psql,... 


## II. Requirements
- Python 3.13+
- `faker` - Generating realistic fake data
- `psycopg2` - psql database adapter
  

## III. Usage
This project using uv (an python package manager):
```bash
uv add faker psycopg2
```
